# Maira's Weekly Meal Plan

A beautiful, animated single-page web app displaying a 7-day meal plan for baby Maira (~11 months).

## Features

- **7-day meal plan** with 5 meals per day (Breakfast, Snack 1, Lunch, Snack 2, Dinner)
- **3D flip card animations** — tap to reveal daily meals
- **Day-by-day navigation** with smooth transitions and swipe support
- **New ingredients guide** with collapsible categories
- **Weight gain tips** and key dietary rules
- Fully **responsive** — works great on phones, tablets, and desktops
- **Keyboard & touch** navigation (arrow keys, swipe gestures)

## Tech

Pure HTML + CSS + JS. No build tools, no frameworks. Just open `index.html`.

## Hosting

Hosted via GitHub Pages. To deploy your own:

1. Fork this repo
2. Go to Settings → Pages → Deploy from branch → `main` / `/ (root)`
3. Your site will be live at `https://<username>.github.io/maira-meal-planner/`
